# python-SQLite-CRUD-WebApp
A Web based SQLite CRUD Application written in Python3.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install streamlit.

```bash
pip install streamlit
pip install pandas
```
